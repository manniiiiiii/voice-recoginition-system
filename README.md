## 项目简介

本项目针对语音识别在噪声环境下准确率下降以及大模型难以移动端部署的问题，设计并实现了一套基于Vosk 轻量化识别引擎与谱减法前端降噪的实时语音识别系统。

系统支持麦克风实时识别和音频文件处理，提供降噪前后结果的左右分屏对比展示，直观呈现降噪算法对识别准确率的提升效果。



## 主要功能

实时语音识别：通过麦克风实时采集语音，低延迟转写

文件批量识别：支持单文件/批量导入 WAV 音频文件

降噪处理：基于谱减法的噪声抑制，可调节开关

对比展示：左右分屏同时显示降噪前后识别结果 

结果导出：识别结果可保存为 Excel/文本文件
